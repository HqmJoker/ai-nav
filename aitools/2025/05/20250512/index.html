✨ 1: GitFriend

GitFriend是一款AI驱动的GitHub助手，旨在通过AI聊天、README生成等功能，简化代码仓库管理，提升开发效率。

GitFriend 是一个基于 React、TypeScript 和 AI 技术的 GitHub 助手，旨在简化 GitHub 仓库的管理，提高效率。它提供以下主要功能：

  • AI 聊天助手： 提供 Git 和 GitHub 相关的帮助，可以解答疑问、解决错误。
  • 动态 README 生成器： 自动生成自定义的 README 文件。
  • Commit Emoji 支持： 通过添加表情符号来改进提交信息的可读性。

地址：https://github.com/krishn404/Git-Friend

✨ 2: Xianyu AutoAgent

Xianyu AutoAgent是闲鱼AI客服机器人，支持7x24小时值守、智能议价和多专家协同。

Xianyu AutoAgent 是一个基于大型语言模型 (LLM) 构建的智能闲鱼客服机器人系统。 它可以实现 24/7 全天候的闲鱼店铺自动化值守，通过智能对话引擎提供智能回复、智能议价、技术支持等服务。

核心功能包括：

  • 智能对话引擎：

    • 上下文感知： 通过轻量级的会话历史存储，能记住完整的对话历史并作为LLM的输入，实现更自然的对话。
    • 专家路由： 基于LLM prompt和规则，能识别用户意图，并动态地将用户分配给议价、技术或客服等不同的“专家”Agent处理。

  • 业务功能：

    • LLM自动回复
    • 上下文管理
    • 阶梯降价策略
    • 网络搜索整合

地址：https://github.com/shaxiu/XianyuAutoAgent

✨ 3: Deep Search Lighting

Deep Search Lighting是一个轻量级、纯Web搜索方案，支持多引擎聚合搜索和结果评估，适用于各类LLM模型。

Deep Search Lighting 是一个轻量级的、纯Web搜索的解决方案，专门为大型语言模型（LLM）设计。它旨在解决传统Web搜索方案的局限性，例如搜索结果质量不高、反射效果不佳、对模型要求高以及小模型难以使用工具等问题。

核心特点：

  • 多引擎聚合搜索： 支持Baidu、DuckDuckGo、Bocha和Tavily等多个搜索引擎，可以聚合搜索结果。
  • 反射策略和可控评估： 提供反射机制，让模型可以自我评估搜索结果的质量。
  • 自定义Pipeline： 适用于所有LLM模型，可以自定义搜索流程。
  • OpenAI风格API兼容性： 易于集成到现有的LLM应用中。
  • 纯模型源码： 方便开发者进行修改和定制。
  • 内置MCP服务器支持： 支持与模型控制平台 (MCP) 集成。
  • 无需框架限制： 可以灵活地嵌入到任何LLM框架中。
  • 速度和结果平衡： 通过可调节的深度参数，平衡搜索速度和结果质量。
  • 小模型友好： 即使是较小的模型也能有效使用。

地址：https://github.com/positive666/deep_search_lightning

✨ 4: HunyuanCustom

HunyuanCustom是腾讯开源的多模态定制视频生成框架，支持图像、音频、视频和文本输入，实现高一致性、可控的视频生成。

HunyuanCustom 是一个腾讯推出的多模态定制视频生成框架，建立在 HunyuanVideo 的基础上。它的核心目标是生成具有指定主体一致性的视频，并且支持多种输入模态，包括图像、音频、视频和文本。

核心特点：

  • 多模态输入： 支持文本、图像（单/多主体）、音频、视频作为条件输入，实现高度灵活的视频定制。
  • 主体一致性： 强调生成视频中人物/对象与输入图像的身份一致性，避免生成“换脸”效果不佳的视频。
  • 应用广泛： 可以用于虚拟人物广告、虚拟试穿、唱歌头像制作、视频编辑（替换视频中的对象）等。

地址：https://github.com/Tencent/HunyuanCustom

✨ 5: 无需代理的LLM网络搜索引擎

这是一个无需代理的LLM网络搜索引擎，支持多引擎搜索、智能解析和异步处理，适用于大模型外部知识调用。

这是一个无需代理的LLM网络搜索引擎，旨在为大型语言模型(LLM)提供强大的外部知识获取能力。它通过以下关键特性实现：

  • 无需代理的网络搜索： 利用Playwright配置国内浏览器支持，无需VPN或代理即可访问主流搜索引擎。
  • 多搜索引擎支持： 支持Bing、夸克、百度、搜狗等，扩大信息来源。
  • 智能意图识别与查询分解： 自动判断用户意图，并将复杂问题分解为多个子任务，提高搜索效率和相关性。
  • 智能体架构： 基于LangGraph框架，构建模块化的智能体链路。
  • 异步并发处理： 提高处理多个搜索任务的效率。
  • 优化内容处理： 包括网页内容切片、重排序和过滤，确保LLM获取高质量、相关的信息。
  • 多端支持： 提供FastAPI后端接口、Gradio Web UI和浏览器插件，方便集成和使用。

地址：https://github.com/itshyao/proxyless-llm-websearch
