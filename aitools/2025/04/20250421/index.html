✨ 1: mcp-agent

mcp-agent是一个基于模型上下文协议(MCP)的轻量级框架，旨在简化构建有效、可组合的智能体应用。

mcp-agent 是一个基于 模型上下文协议 (MCP) 的简单且可组合的框架，用于构建智能代理。它简化了代理的构建过程，使开发者可以专注于核心业务逻辑。

主要特点：

  • 基于 MCP： 专门为 MCP 设计，保证与其他 MCP 服务无缝集成。
  • 轻量级框架： 更像是一个代理模式库，而不是一个臃肿的框架。
  • 可组合性与可定制性： 实现了定义良好的工作流，并且这些工作流可以组合，形成更复杂的工作流。支持对模型提供商、日志记录、编排器等进行全面定制。
  • 程序化控制流： 开发者只需要编写代码，而不是考虑图、节点和边。使用 if 语句进行分支逻辑，使用 while 循环进行循环。
  • 人工输入和信号： 支持暂停工作流以获取外部信号，例如人工输入，这些信号作为代理可以调用的工具公开。

地址：https://github.com/lastmile-ai/mcp-agent

✨ 2: DocAgent

DocAgent是一个利用多智能体和分层处理，为Python代码库自动生成高质量文档字符串的系统。

DocAgent 是一个利用多智能体和分层处理方法，为 Python 代码库自动生成高质量、上下文感知型文档字符串（docstring）的系统。

核心思想：

  1. 分层遍历：DocAgent 从依赖较少的文件开始，分析代码组件的依赖关系，逐步构建文档基础，然后再处理更复杂的代码。
  2. 智能体系统：DocAgent 使用由Reader（阅读器）、Searcher（搜索器）、Writer（编写器）和Verifier（验证器）等专门智能体组成的团队，由Orchestrator（协调器）协调。该系统收集上下文信息（内部和外部的），根据标准起草文档字符串，并以迭代方式验证其质量。

地址：https://github.com/facebookresearch/DocAgent

✨ 3: Google Search Console MCP server for SEOs

Google Search Console MCP server 连接 GSC 和 Claude AI，让 SEO 通过自然语言对话分析网站数据，提升优化效率。

Google Search Console MCP server for SEOs 是一个工具，它连接了 Google Search Console (GSC) 和 Claude AI。 通过与 Claude 的自然语言对话，SEO 专业人士可以分析他们的 SEO 数据。 它提供了一系列功能，包括：

  • **属性管理：**查看和管理 GSC 属性，添加新网站，删除网站。
  • 搜索分析与报告： 发现带来访客的搜索查询，跟踪展示次数、点击次数和点击率，分析性能趋势。 能够通过Claude生成图表和图形来可视化数据。
  • **URL 检查与索引编制：**检查特定页面是否存在索引问题，查看 Google 上次抓取页面的时间，并获取改进索引编制的可行建议。
  • **站点地图管理：**查看所有站点地图及其状态，提交新的站点地图，检查站点地图中的错误或警告。

地址：https://github.com/AminForou/mcp-gsc

✨ 4: Google Ads MCP

Google Ads MCP: 连接Google Ads和Claude AI，通过自然语言对话分析广告数据，优化广告效果。

Google Ads MCP (MCP for Google Ads) 是一个工具，它连接了 Google Ads 和 Claude AI，允许用户通过自然语言对话分析他们的广告数据。 这个工具通过与 Claude AI 聊天，提供对广告系列信息、效果指标、关键词分析和广告管理的访问。

核心功能：

  • 账号管理: 集中查看和管理所有 Google Ads 账号。
  • 广告系列分析与报告: 分析广告系列表现，追踪各项指标，进行时间段对比，并用图表可视化数据。
  • 关键词与广告效果分析: 识别表现优异和欠佳的关键词，分析广告文案的效果，并提供改进建议。
  • 预算与出价管理: 监控广告系列预算和支出，分析出价策略，识别优化机会。
  • GAQL 查询: 运行自定义 GAQL 查询以获取特定的数据。

地址：https://github.com/cohnen/mcp-google-ads

✨ 5: Find My Kids

Find My Kids利用WhatsApp和DeepFace，检测照片中的特定人脸并通知指定联系人。

Find My Kids 是一个基于 WhatsApp 的人脸识别机器人，它使用 Green API 和 DeepFace 技术，在 WhatsApp 群组中检测预先设定的面孔，并立即通知指定联系人。

主要功能：

  • 人脸识别: 利用 DeepFace 识别特定人员。
  • WhatsApp 集成: 通过 Green API 接入 WhatsApp。
  • 实时通知: 在群组图片中发现目标人物时，立即通知指定联系人。

地址：https://github.com/t0mer/Find-my-kids
