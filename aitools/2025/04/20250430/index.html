✨ 1: ACI

ACI是开源基础设施，为AI智能体提供600+工具的统一访问，并支持MCP服务器。

ACI (ACI.dev) 是一个开源的基础设施层，旨在为AI Agent的工具使用提供支持。它提供以下核心能力：

  • 统一的工具访问: 让AI Agent能够通过统一的方式访问600+个工具，无需为每个工具单独编写认证流程和API客户端。
  • 多租户认证和权限管理: 内置OAuth认证流程和密钥管理，支持开发者和最终用户。
  • 动态工具发现: AI Agent能够动态地发现和使用合适的工具。
  • 两种访问方式: 提供统一的模型-上下文-协议 (MCP) 服务器，或者轻量级的 Python SDK，兼容任何 LLM 框架。
  • 100% 开源: 包括后端、开发者门户和集成在内的所有组件都以 Apache 2.0 许可证发布。

地址：https://github.com/aipotheosis-labs/aci

✨ 2: iOS Simulator MCP Server

iOS Simulator MCP Server是一个模型上下文协议服务器，用于与iOS模拟器交互，获取信息、控制UI和检查UI元素。

iOS Simulator MCP Server 是一个模型上下文协议（MCP）服务器，用于与 iOS 模拟器进行交互。它允许你通过获取模拟器信息、控制 UI 交互以及检查 UI 元素来与 iOS 模拟器进行通信。

主要特点：

  • 获取当前启动的 iOS 模拟器的 ID。
  • 与模拟器 UI 交互，包括：
    • 描述屏幕上的所有辅助功能元素。
    • 在屏幕坐标上点击。
    • 输入文本。
    • 在坐标之间滑动。
    • 获取特定坐标处 UI 元素的信息。
    • 截取模拟器屏幕截图。
  • 使用环境变量过滤特定工具。

地址：https://github.com/joshuayoes/ios-simulator-mcp

✨ 3: Muyan-TTS

Muyan-TTS是一个预算5万美元的播客TTS模型，预训练超10万小时音频，支持高质量零样本语音合成和说话人适配。

Muyan-TTS 是一款为播客应用设计的可训练的文本转语音 (TTS) 模型，总预算约为 5 万美元。它基于超过 10 万小时的播客音频数据进行预训练，从而实现高质量的零样本 TTS 合成。此外，Muyan-TTS 支持使用几十分钟的目标语音进行说话人自适应，使其能够针对个人声音进行高度定制。

地址：https://github.com/MYZY-AI/Muyan-TTS

✨ 4: GraphGen

GraphGen利用知识图谱生成高质量合成数据，优化大型语言模型的监督微调效果。

GraphGen是一个基于知识图谱的合成数据生成框架，用于增强大型语言模型（LLMs）的监督微调效果。 它通过以下步骤运作：

  1. 知识图谱构建: 首先，GraphGen从原始文本中构建一个细粒度的知识图谱。
  2. 知识缺口识别: 使用期望校准误差指标，GraphGen识别LLMs中的知识缺口，并优先生成针对高价值、长尾知识的问答对。
  3. 多跳邻域采样: 通过多跳邻域采样，捕获复杂的关系信息，增加生成的QA数据多样性。
  4. 风格控制生成: 使用风格控制技术，进一步多样化生成的QA数据。

地址：https://github.com/open-sciencelab/GraphGen

✨ 5: Deepwiki MCP Server

Deepwiki MCP Server是一个非官方工具，它通过MCP抓取Deepwiki页面，转换为Markdown格式并返回文档。

Deepwiki MCP Server 是一个非官方的工具，它通过 MCP 协议接收 Deepwiki 网站的 URL，然后：

  1. 抓取 Deepwiki 网站上与该 URL 相关的页面。
  2. 转换 抓取到的页面内容为 Markdown 格式。
  3. 返回 一个整合的 Markdown 文档，或者按页面返回 Markdown 列表。

地址：https://github.com/regenrek/deepwiki-mcp
