✨ 1: abogen

Abogen是一款强大的文本转语音工具，支持多种格式，并可生成同步字幕，效果自然。

Abogen是一个强大的文本转语音工具，可以快速将EPUB、PDF或文本文件转换为高质量的音频，并生成配套的字幕。 它使用Kokoro-82M模型，生成的语音自然流畅，适合制作有声读物、为Instagram、YouTube、TikTok等平台制作配音，以及其他需要自然语音合成的项目。

地址：https://github.com/denizsafak/abogen

✨ 2: CAD-MCP Server

CAD-MCP Server是能通过自然语言控制主流CAD软件绘图的创新服务，支持多种绘图和NLP功能。

CAD-MCP Server 是一个创新的 CAD 控制服务，它允许用户通过自然语言指令来控制 CAD 软件，进行绘图操作。它结合了自然语言处理和 CAD 自动化技术，用户无需手动操作 CAD 界面，只需通过简单的文本命令即可创建和修改 CAD 图纸。它支持多种 CAD 软件（AutoCAD, GstarCAD, ZWCAD），并提供包括画线、画圆、画弧、矩形、多段线、添加文本、图案填充、尺寸标注以及图层管理和保存等基本绘图功能。

地址：https://github.com/daobataotie/CAD-MCP

✨ 3: ChatWiki

ChatWiki是开源的知识库AI问答系统，支持多种文档格式和国产大模型，企业可快速搭建私有问答系统。

ChatWiki 是一款开源的知识库 AI 问答系统。 它基于大语言模型 (LLM)、检索增强生成 (RAG) 以及 GraphRAG 知识图谱构建，旨在帮助企业、高校和政府部门快速搭建私有的 AI 知识库问答系统。ChatWiki 提供了开箱即用的数据处理和模型调用能力，支持多种文档格式（如 docx、excel、pdf、ofd 等），并且兼容国产化浏览器，支持 Deepseek、千问、豆包等 20 多个大模型。

地址：https://github.com/zhimaAi/chatwiki

✨ 4: llm.pdf

llm.pdf是一个概念验证项目，展示了在PDF文件中运行完整大型语言模型(LLM)的可能性。

llm.pdf 是一个概念验证项目，它展示了在 PDF 文件中运行完整的大型语言模型 (LLM) 的可能性。

工作原理:

该项目使用 Emscripten 将 llama.cpp 编译成可在 PDF 文件中执行的 JavaScript 代码。 同时，它将整个 LLM 模型文件以 Base64 编码的形式嵌入到 PDF 文件中。 最终，当 PDF 文件被打开时，它会运行嵌入的 JavaScript 代码，该代码会解码 LLM 模型并使用 llama.cpp 进行推理。

地址：https://github.com/EvanZhouDev/llm.pdf

✨ 5: Spotify MCP Server

Spotify MCP Server 是一个轻量级服务器，让 AI 助手能控制 Spotify 播放和管理播放列表。

Spotify MCP Server 是一个轻量级的 Model Context Protocol (MCP) 服务器，它的主要作用是让 AI 助手（如 Cursor 和 Claude）能够控制 Spotify 的播放和管理播放列表。 换句话说，它充当了 AI 助手和 Spotify 之间的桥梁，允许 AI 通过调用预定义的工具来执行各种 Spotify 操作。

地址：https://github.com/marcelmarais/spotify-mcp-server
